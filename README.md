# Custom c++ Singly Linked List Data Structure

بما اني صار لي فترة ماكتبت اي كود ومابعد اتعلم الداتا ستركشرز, قلت لازم ابدا اتعلمه. ومن طرقي المفضلة للتعلم اني احاول ابنيه بنفسي.
الصراحة واجهت صعوبات بحكم اني جديد على c++
لكن كانت تجربة ممتعة وعلمتني الكثير خصوصا الان حتى السبيس كومبليكسيت فاهم ليش كذا لاني انا اللي بنيت هالشي

*للتنويه* انا كتبت هلكود في وقت الفراغ بوسط الاختبارات وللان مو كامل وفي فكرة اني اي داتا ستركشر اتعلمه بحطه بالريباستوري وبزيد اوامر زيادة.
ايضا حاليا بس يدعم انتيقر ف مستقبلا ان شاءالله بضيف باقي الانواع.

## الاوامر الموجودة حاليا
* **insertathead** يستخدم اذا بتضيف رقم بالقائمة تحط الرقم بالانديكس
* **deletevalue** اذا فيه رقم معين تبي تحذفه من القائمة تكتب الرقم بالانديكس ويحذفه لك
* **printlist** لطباعة القائمة كاملة
* **sizeoflinkedlist** هذا يرجع لك رقم انتيقر لحجم القائمة لا اكثر
