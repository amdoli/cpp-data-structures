# Custom c++ Singly Linked List Data Structure

بما اني صار لي فترة ماكتبت اي كود ومابعد اتعلم الداتا ستركشرز, قلت لازم ابدا اتعلمه. ومن طرقي المفضلة للتعلم اني احاول ابنيه بنفسي.
الصراحة واجهت صعوبات بحكم اني جديد على c++
لكن كانت تجربة ممتعة وعلمتني الكثير خصوصا الان حتى السبيس كومبليكسيتي فاهم ليش كذا لاني انا اللي بنيت هالشي

*للتنويه* انا كتبت هالكود في وقت الفراغ بوسط الاختبارات وللان مو كامل وفي فكرة اني اي داتا ستركشر اتعلمه بحطه بالريباستوري وبزيد اوامر زيادة.
ايضا حاليا بس يدعم انتيقر ف مستقبلا ان شاءالله بضيف باقي الانواع.

## الاوامر الموجودة حاليا (linkedlist.h)
* **insert** يستخدم اذا بتضيف رقم بالقائمة تحط الرقم بالانديكس
* **deleteValue** اذا فيه رقم معين تبي تحذفه من القائمة تكتب الرقم بالانديكس ويحذفه لك
* **printList** لطباعة القائمة كاملة
* **sizeOfLinkedList** هذا يرجع لك رقم انتيقر لحجم القائمة لا اكثر
* **showTheTop** بما ان الhead داخل البرايفت نستخدم هالامر عشان نطلع القيمة للهيد

## البرامج اللي سويتها
كان هدفي من البداية اني اسوي math parser الا وهو انك تحط معادلة بفنكشن وهو يحله لك. 
* **tokenizer.cpp** يسوي للمعادلة توكنايز اوتوماتيك عنك
* **Parser.cpp** يرتب المعادلة باستخدام القورذم ( shunting yard algorithm )
* **Evaluator.cpp** يحل المعادلة المعاد تريبها من الاقورذم المذكور اعلاه

اذا تبي تستفيد من الmath parser لازم تسوي include للبرامج المذكورة وحاولت اوضح كيفية استخدامه ببرنامج ال main.cpp
